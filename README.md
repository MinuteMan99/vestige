# Vestige

Vestige is an interactive Nigerian history application designed to help users explore historical facts, traditions, events, and cultural evolution in creative and engaging ways.

The application presents history through visual storytelling, comparisons, timelines, galleries, and cultural preservation pages, making learning about Nigeria’s past more immersive and accessible.

---

## Features

### Then vs Now
Compare past and present Nigerian lifestyles, traditions, technology, fashion, transportation, and culture.

### Historical Timeline
Explore major events across different eras of Nigerian history, including:

- Pre-Colonial Era
- Colonial Era
- Independence Era
- Digital Age
- Future Age

### Lost Traditions
Discover traditions, practices, and cultural elements that have gradually become forgotten or lost over time.

### Historical Gallery
Browse through historical images and visual content with cultural and historical significance.

### Interactive Navigation
Smooth page navigation system handled through a centralized `main.py` controller.

## Installation

### 1. Clone the Repository

```bash
git clone <repository-link>
```

### 2. Navigate Into the Project Directory

```bash
cd vestige
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

Open the `main.py` file inside the Vestige repository and run it.

```bash
python main.py
```

---

## How the Application Works

The `main.py` file acts as the main controller for the application.

It manages:
- Navigation between pages
- Displaying different page classes
- Command-based page switching
- Launching the initial welcome page

The application starts on the Welcome Page before allowing users to explore other sections of the app.
